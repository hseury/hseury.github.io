ViewPager的事件拦截机制

onInterceptTouchEvent没有down事件，ontouchevent有move事件?

首先是一段log：

log打在这个viewpager的dispatchTouchEvent,onInterceptTouchEvent和onTouchEvent方法中。result表示该方法的返回值。

按照google官方文档的说明：

如果onInterceptTouchEvent方法返回true，那么它将不会收到后续事件，事件将会直接传递给目标的onTouchEvent方法（其实会先传给目标的onTouch方法）
如果onInterceptTouchEvent方法返回false，那么所有的后续事件都会先传给onInterceptTouchEvent，然后再传给目标的onTouchEvent方法。

但是，为什么我们在onInterceptTouchEvent方法中返回false之后，却收不到后续的事件呢？？通过实验以及stackoverflow上面的一些问答得知，当我们在onInterceptTouchEvent（）方法中返回false，且View的onTouchEvent返回false的情况下，onInterceptTouchEvent方法才会收到后续的事件。

将上面程序的onTouchEvent方法强制返回false，得到下面的log。

可见，在当前view，dispatchTouchEvent每次手势滑动都会传递，onInterceptTouchEvent返回false，表示不拦截事件，则ontouchevent不调用，onInterceptTouchEvent返回true，表示拦截事件，onTouchEvent会处理，但是如果第一次onTouchEvent返回true，则以后的onInterceptTouchEvent都不会被调用，因为view已经确认要处理事件了。

虽然这个结果与官方文档的说法有点不同，但实验说明是正确的。仔细想想这样的逻辑也确实非常合理：因为onInterceptTouchEvent方法是用来拦截触摸事件，防止被View捕获。那么现在View在onTouchEvent中返回fTrue，明确声明自己会处理这个触摸事件，那么这个时候就不需要拦截了，因此onInterceptTouchEvent不需要拦截这个事件，那也就没有必要将后续事件再传给它了。

因此，在dispatchTouchEvent中可以获得所有事件，其他两个方法不会每次事件传递都会调用。

persistent 为true，预装时候crash，报找不到类的问题

一般没有persist属性的应用在进行应用自升级的时候，系统会保证程序运行的状态是OK的，因为在安装应用（应用自升级）的过程中，系统会KILL此应用的进程，同时清理之前在AMS（ActivityManagerService）中记录的此应用进程中的各种组件（Activity，Service,Receiver,ContentProvider），在下次该新版本的应用重新启动时，系统会重新new一个进程并重新加载各种组件运行，所以不会有问题；当应用有persist属性并且生效时（不但要添加persist标签，还要是系统签名或者有系统权限），系统安装新应用的过程与安装不带此属性的过程不同，主要在是因为：系统不会KILL此应用的进程。系统不会清理之前在AMS（ActivityManagerService）中记录的各种组件的信息，只会把新版应用中的各种组件记录到AMS中。这样的话如果前后2个版本的应用的某个组件（如Actvivity）名称没有变化，但是内部增加了很多新的类和新的逻辑实现，就会导致加载的组件有的是新版本的有的是老版本，导致程序功能错乱。

可以参考Android应用具有persist属性时如何自升级

Android中为什么主线程不会因为Looper.loop()里的死循环卡死？

查看了答案之后，自己总结了下。

在ActivityThread类中，

其中H是ActivityThread对应的Handler,它的handleMessage()方法是这样的

Activity的生命周期都有对应的case条件了，ActivityThread有个getHandler() 方法，得到这个handler就可以发送消息，然后loop 里就分发消息，然后就发给handler，然后就执行到 H(Handler)里的对应代码。所以这些代码就不会卡死，有消息过来就能执行。所有的操作都是执行在Looper中的。Looper.loop()方法中，如果next消息为null，就会return，此时确实会进入block状态，等到有消息时，会被再次唤醒，继续依次执行消息。

Android 的应用是由事件驱动的，looper.loop() 不断地接收事件、处理事件，每一个应用都是运行在 Looper.loop() 的控制之下，如果它停止了，应用也就停止了。只能是某一个应用阻塞 Looper.loop()，而不是 Looper.loop() 阻塞应用。

这是一种常见的事件驱动模型，windows的消息循环机制也类似，可以深入了解这些知识epoll+pipe。

原子操作

原子操作是指不会被线程调度机制打断的操作；这种操作一旦开始，就一直运行到结束，中间不会有任何 context switch。原子操作可以是一个步骤，也可以是多个操作步骤，但是其顺序不可以被打乱，也不可以被切割而只执行其中的一部分。将整个操作视作一个整体是原子性的核心特征。

使用条件对象（条件变量）来管理哪些已经获得了锁，但是却不能做有用工作的线程。

一个锁可以有一个或者多个相关的条件对象。

java中long和double的非原子性

在java中，long和double类型的数据都占8个字节长度，即64位。对于long和double类型的读写操作，java并不能保证其原子性。

For the purposes of the Java programming language memory model, a single write to a non-volatile long or double value is treated as two separate writes: one to each 32-bit half. This can result in a situation where a thread sees the first 32 bits of a 64-bit value from one write, and the second 32 bits from another write.
Writes and reads of volatile long and double values are always atomic.
Writes to and reads of references are always atomic, regardless of whether they are implemented as 32-bit or 64-bit values.
Some implementations may find it convenient to divide a single write action on a 64-bit long or double value into two write actions on adjacent 32-bit values. For efficiency’s sake, this behavior is implementation-specific; an implementation of the Java Virtual Machine is free to perform writes to long and double values atomically or in two parts.
Implementations of the Java Virtual Machine are encouraged to avoid splitting 64-bit values where possible. Programmers are encouraged to declare shared 64-bit values as volatile or synchronize their programs correctly to avoid possible complications.

总结一下：

  1. 在java中，非volatile的double和long类型的写操作被分成了两个32位的写操作，所以对于64位的double和long类型的写操作就不是原子操作。
  2. 对于volatile类型的long和double类型的写操作是原子操作。
  3. 对于引用类型的读写操作都是原子操作，无论引用类型的实际类型是32位的值还是64位的值。
  4. 在我们多线程情况下用到long和double的时候，尽量使用volatile或者同步对于数据操作进行控制。

在多线程环境下对int和long进行赋值和读取操作，都会出现线程安全问题，那么有啥不同呢？

  1. 对于int类型的不恰当操作会造成的问题是：老的数据会被读取到。
  2. 对于long或double类型的不恰当操作会造成的问题是：可能从未出现过的数据被读取到。

虽然int型是原子数据，但是并不保证一个线程写入的值，对于另一个线程是可见的。
所以需要同步操作。

读方法和写方法都需要同步。如果不需要互斥，只需要通信，考虑volatile。

圣彼得堡---我的旅途1/2

发表于 2016-10-07 | 分类于 travel | 阅读次数

启程

在莫斯科暴走一整个白天之后，我已经相当疲劳了。凌晨坐上莫斯科通往圣彼得堡的夜间火车后，我发现小小车厢内四个人，就我一个是亚洲人。我的上铺是个高大的商务人士（感觉），对面是一个约三十多岁的妈妈和她的小孩，乘务员大妈说了一大堆俄语，我完全听不懂，还是感谢这位妈妈为我翻译成英语，否则我又会再一次体会到语言不通带来的绝望。火车开动之后，我很快进入了梦乡，再一次醒来已经是凌晨6点多，窗外已经开始泛白，我拿出手机看了看，毫无信号，此时的火车就像是一艘在大海上航行的渔船。我想起了很多年以前小学语文课本里的一些课文，窗外倒走的树林，泛出鱼肠白的天际线这些意象一下子填满了我的脑袋。火车到站，走出车厢之后，就被圣彼得堡潮湿的空气刺激到了，天气也和莫斯科的阴沉不同，天空好蓝好漂亮，搭乘地铁，存放完行李之后，就是第一站——冬宫。

冬宫

冬宫全称艾尔米塔什博物馆，是世界四大博物馆之一，与巴黎的罗浮宫、伦敦的大英博物馆、纽约的大都会艺术博物馆并称世界四大博物馆。该馆最早是叶卡捷琳娜二世女皇的私人宫邸，位于俄罗斯圣彼得堡涅瓦河畔。博物馆中有1.5万幅绘画，1.2万件雕塑，60万幅线条画作品，100万块硬币和证章，22.4万件古代家具、瓷器、金银制品、宝石与象牙工艺品等。这些工艺品分别陈列在350多个展厅。
350个展厅展示了占艾尔米塔什全部珍宝百分之五的收藏品。虽然只有百分之五，一分钟看一件，不眠不休也得花上一个季度才能看完。达·芬奇存世作品只有十幅，艾尔米塔什就拥有两幅。

接下来是走马观花的图。

冬宫最美的走廊——拉斐尔长廊。俄国女皇叶卡捷琳娜二世，在1775年参观过梵蒂冈的拉菲尔壁画之后，兴起复制壁画的念头，于是派遣画师前往当地临摹，将整座走廊都复制重现在冬宫。因工程浩大，直到1792年才全部完工，历经十七年。

圣·乔治大厅也叫御座大厅，在冬宫的心脏地带。这个御座大厅建造于叶卡捷琳娜二世（1762-1796）在位期间的1787-1795年，这间别具风格的大厅，完全以严格的古典主义风格建造而成。这个厅的地板和天花板都是一一对称的，实在是让人震惊。

收藏油画最大的厅，就属这座意大利文艺复兴天窗厅了。

冬宫实在是个艺术的宝库，漫步其中体会到的是无与伦比的震撼，中学历史课本里学习过的一些知识从记忆深处被唤醒，让我此刻感受到从前的经历并非不值一提，在人生的某一个时刻，它会被重新唤起，这一切都不是没有意义的。我后悔以前对这些东西没有重视，如果我对俄国的历史更加熟悉，我会更加享受这次的参观。

这次冬宫之旅我也开始意识到，中国的旅游团真多，中国的旅游团真喜欢拍照，相比较而言，日本的游客是比较让人敬佩的，他们很少拍照，一幅油画前可以停留很久，明显可以感受到，他们是真的来参观博物馆的。

枢密院广场

枢密院广场是住所和冬宫前的一个小公园，也是我放下行李第一眼看到的圣彼得堡景点，天空蓝得纯净，阳光下的顶泛着金色的光芒。呼吸着圣彼得堡潮湿的空气，我强烈的感受到了另一种美。广场的地面是柔软的湿土，脚踩下去会感到轻微的弹性，还会有砂石摩擦的簌簌的声音。这是我在圣彼得堡最常去的地方，每天都会在长椅上坐上一会儿。

枢密院广场的不远处，位于俄罗斯圣彼得堡在伊萨基辅大教堂与涅瓦河之间，有”十二月党人”广场·广场中央矗立着彼得大帝的纪念碑，这就是著名的”青铜骑士”雕像，它高5米，重20吨。
彼得大帝骑在腾跃的骏马上,精神坚定。马的后蹄踩着一条毒蛇,象征打败敌人。这座纪念碑于1782年8月7日彼得大帝登基100周年的时候揭幕.纪念碑的基石上刻着:”献给彼得大帝,叶卡捷琳娜二世于1782年夏.”普希金在自己的一篇叙事诗中将这座纪念碑称为”青铜骑士”,从此以后,”青铜骑士”就成了彼得大帝纪念碑的代名词.

圣以撒大教堂

圣以撒大教堂是圣彼得堡最大的俄罗斯东正教堂，圣以撒大世界上第二大的东正教教堂，仅次于莫斯科的救世主大教堂。高度达到101.5米，是以彼得大帝的主保护圣人——达尔玛蒂亚的圣以撒命名的，始建于1818年，前后花了40年的时间于1858年建成。是由俄罗斯当时的沙皇亚历山大一世命令建造的，采用了新古典主义的建筑风格，是根据法国出生的建筑家奥古斯特·蒙特费兰德的设计方案经修改后建造的。蒙特费兰德在40年间里负责监造这座大教堂，1858年建成时，这位建筑师也离开了人世。十月革命之后，由于它是皇权的象征，停止了宗教活动，一度曾废弃，后改作宗教史及无神论的博物馆。

登上顶部长廊，就可以把圣彼得堡的城市风光尽收眼底。

夏宫

彼得大帝夏宫(Peter the Great’s Summer Palace)，又称“俄罗斯夏宫”、“彼德宫”。它位于芬兰湾南岸的森林中，它距圣彼得堡市约30公里，占地近千公顷，由瑞士人多梅尼克·特列吉尼设计，是历代俄国沙皇的郊外离宫，也是圣彼得堡的早期建筑。夏宫分为上花园和下花园，大宫殿在上花园。 内外装饰极其华丽，两翼均有镀金穹顶，宫内有庆典厅堂。 礼宴厅堂和皇家宫室。 大宫殿前是被称作大瀑布的喷泉群。 这里有37座金色雕像，29座潜浮雕，150个小雕像，64个喷泉及2座梯形瀑布。 在喷泉群一个大半圆形水池的中央，耸立着大力士参孙和狮子搏斗的雕像，这就是著名的隆姆松喷泉。参孙双手把狮子的上下颚撑开，泉水从狮子口中冲天而出，水柱直喷22米之高。这一雕塑象征着俄罗斯战胜瑞典。喷泉群在蓝天白云下，闪耀着鎏金的光辉和水流的银影。黄金色的雕塑沐浴在水幕中，玉珠飞溅，光彩夺目。漫步在幽静高大的森林中，每走一小段路，眼前就会闪烁出飞溅的玉珠。

大力士参孙和狮子搏斗的喷泉十分符合我的没血，肌肉喷张，展现了男子汉的力与美。

在下花园漫步，可以感受到俄罗斯人变态的对称性，以隆姆松喷泉为轴心，两边的布局基本一毛一样，也就是说，这个花园参观一半就可以了……

滴血大教堂

滴血大教堂是圣彼得堡为数不多的传统式东正教堂，教堂上有五光十色的洋葱头顶，反映了俄国十六和十七世纪的典型的东正教教堂建筑风格。教堂轮廓美丽，装饰花花绿绿，与古老俄罗斯风格与附近的古典式的建筑物成鲜明对比。

教堂高度约81米，宽阔的外形，采用了与莫斯科巴克洛夫教堂相同的构造，与华西里教堂的区别在于镶嵌复杂、颜色艳丽的影像图案，用丰富的彩色图案瓷砖、搪瓷青铜板装饰面。

叶卡捷琳娜宫

叶卡捷林娜宫又称沙皇村，是叶卡捷琳二世女皇修建的。宫殿的格局则是建筑精巧淫靡，色彩清新柔和，弥漫着女性的柔美、娇媚的风韵。可以说园中到处是诗，到处是画，无处不飘动着令人心醉的旋律，无处不弥漫着花草的芬芳，女皇生前声色犬马、骄奢淫靡的气息依然沁淫着整座园林。天蓝色的外表耀眼夺目，洋溢着喜庆气氛，造型丰富的雕塑和凹凸有致的结构使数百米长的建筑丝毫不显得单调呆板。皇宫教堂那五个圆葱头式尖顶在碧空下金光灿灿，几乎从园内任何地方都远远望得见。

琥珀屋本身就是个传奇。它是1716年由普鲁士国王威廉一世送给俄罗斯彼得大帝的礼物，墙面镶嵌有6吨多的琥珀和名贵珠宝，闪耀着从柠檬黄到金红色，人类可以想象的黄色系中一切的色彩，辉煌得令人窒息，被列为世界第八大奇迹。“二战”期间，这座宝屋被纳粹劫掠窃取。直到圣彼得堡300年（2003年）纪念的时候，才由俄国巧匠重新复原出了当时的面貌。不过由于这件展厅不允许拍照……所以没图……

值得一提是宫殿门前的花园，门票只要十块钱人民币，确实我这辈子见过最美的花园。在这里还有活生生的松鼠跳来跳去，人与自然十分和谐。漫步其中，我再也不想举起手中的相机，只想好好地在这里坐着，看风景，呼吸空气，同时在想要是我家医生这时候坐在我旁边该有多好。

彼得要塞

这座堡垒由彼得大帝创建于1703年儒略历5月16日（公历5月27日），位于涅瓦河北岸的兔子岛（涅瓦河三角洲最上游的一个岛）。

在大北方战争期间，建此要塞是为了防备瑞典袭击新首都。该堡垒从来没有实现其军事目的。此要塞包括六个堡垒，1706至1740年用石头重建。

涅瓦桥日落和战神广场

离开彼得要塞，在涅瓦桥上遇到了壮丽的日落，在轰隆的大桥上吸了好久的尾气，拍了好久照片……

走过涅瓦桥之后就是战神广场了，北面是大理石宫，南部边界是莫伊卡河。它得名于罗马战神马尔斯，占地面积约9公顷，是圣彼得堡最古老、最美丽的广场之一。广场上栽有大量的丁香花，竖立着俄罗斯著名的苏沃洛夫统帅的塑像，是根据罗马战神马尔斯的形象建造的，因此又称马尔索洛校场。广场一侧是巴普洛夫斯基兵营，兵营大楼宽数百米，雄伟壮观。广场中心是革命英雄纪念碑，从1917年起，“长明火”一直燃烧至今，莫斯科克里姆林宫旁的无名烈士墓的“长明火”的火种也取自这里。以纪念碑为中心，向不同方向延伸出数条林荫道，林荫道之间是点缀着灌木丛的草坪，整个广场处于椴木林和草坪的怀抱之中。可能很多攻略上都不会把这里作为景点，但是我无意间走到这里，在这里我能感受到勇气、历史还有尊重。

言归正传，在飞机上断断续续睡了十多个小时之后，我终于踏上了俄罗斯的土地，然鹅，排队出关慢的生无可恋，漫长的2小时排队之后，我终于出关，见到了正在毛国留学的梁博，感激梁博在机场等了我2个多小时，还把我送到订的旅馆，没有他我简直在莫斯科活不下去，语言不通去俄罗斯旅游真是史诗难度，放好行李之后，这时候已经是晚上9点多了，梁博带我去商场吃了一顿好的，果酱特别好喝，其他的简直吃不下去，我看了四周，我说这里都坐不满，开不了多久就会关门吧，梁博说，这家店的生意已经很好了，都快坐满了……

吃完去超市里买了一些需要的东西，梁博还特别热心地告诉我这里哪些店可以喝咖啡哪些店可以吃饼，我一一记下了，不过这没有什么卵用，第二天早上我直接就迷路了，任何店都没有找到~

莫斯科河 & 高等经济学院

我住的地方就在莫斯科河附近，这条河是莫斯科的母亲河，此时的莫斯科已经是深秋，空气异常潮湿，金黄的树叶飘了一地，非常好看。

街道上人真的很少，很有可能一条路上就只有你自己一个人还有无数只吃得太饱，正在散步消食的鸽子。

梁博的宿舍楼让我想起了我还在彰武的时光，一个过道，两排房间，梁博的房间靠南，有一排明亮的窗户，窗外是高大的树木，还有调皮的小鸟在上面叽叽喳喳。每层楼还有一个自习教室，坐在里面，喝着梁博炮制的咖啡，我好像又回到了校园时光，午饭和梁博，梁博的室友匈牙利队长，导游台湾妹子姿懿一起在厨房吃了大盒披萨，看着这群留学生来自不同国家，巴拉巴拉的聊天吃饭，我好羡慕他们，要是自己当时出来读个书该多好。吃完披萨我已经非常困了，在梁博床上坐了会儿直接就睡着了。

麻雀山 & 莫斯科大学

这一天的游览从麻雀山开始，麻雀山位于莫斯科西南，是莫斯科的最高处，最高处海拔高度为220米，莫斯科河从山脚流过。在麻雀山山顶上有观景台，观景台正对莫斯科大学的正门，由此可俯瞰莫斯科河与莫斯科市区，美丽景色尽收眼底。不过这天天气真的很差，全是白板天，而且也很冷，在寒风中瑟瑟发抖的我实在是没能欣赏俯瞰莫斯科的美。

从观景台走下来，就是莫斯科大学了，它是俄罗斯联邦规模最大、历史最悠久的综合性高等学校，建于1755年，已经有200多年历史了，而它的主楼正是著名的莫斯科斯大林式建筑群——“七姐妹”之一。 莫斯科大学远远看去只能用”大”,”好大”之类的形容词来形容，当然比绝对面积，是肯定比不过国内的大学的，但这个大学是个封闭的碉堡，不是学生没法进入，在墙外看它的墙壁高高在上，四个钟楼守护的这个学校，分别代表了时间，气压，温度，还有一个实在没看到，在城楼下面瞻仰这个大学，内心os：”碉堡了”。

在莫大旁边，铺满了金色落叶的学生宿舍可能是我此生见过最美的学生宿舍了。

高尔基公园 & 救世主教堂

离开莫斯科大学之后，坐地铁到了高尔基公园，位于普希金海滨沿岸大街和列宁大街之间的位置。这个公园正在翻修，喷泉也没开，没什么好看的。有意思的是我在公园门口拍照的时候，一个人走上前找我说话，我当场就懵逼了，在这里，我才是辣个外国人，姿懿耐心的听完告诉我那个人是想我为他负责的XX保护协会捐点钱……那你眼神真诚点好不好~

救世主教堂是世界上最高的东正教教堂，也是最大的东正教教堂之一。该教堂是拿破仑战争后，在1812年12月25日由沙皇亚历山大一世下令修建的，其目的是为了感谢救世主基督“将俄罗斯从失败中拯救出来，使她避免蒙羞”，并纪念在战争中牺牲的俄罗斯人民。

在教堂前面可以远眺远方的克里姆林宫。

红场

红场是俄罗斯首都莫斯科市中心的著名广场，位于莫斯科市中心，西南与克里姆林宫相毗连。原是前苏联重要节日举行群众集会和阅兵的地方。辟于十五世纪末，十七世纪后半期取今名。平面长方形，面积约4公顷。西侧是克里姆林宫，北面为国立历史博物馆，东侧为百货大楼，南部为瓦西里布拉仁教堂。临莫斯科河。列宁陵墓位于靠宫墙一面的中部。墓上为检阅台，两旁为观礼台。地面很独特全部由条石铺成，显得古老而神圣。红场是莫斯科历史的见证，也是莫斯科人的骄傲。在俄语中，“红色的”含有“美丽”之意，“红场”的意思就是“美丽的广场”。

红场北面，是一座三层红砖白顶楼，其式样仿照古代俄罗斯建筑，南北各有尖塔8座。1995年5月8日，为纪念二战胜利50周年，立起了二战英雄朱可夫元帅的雕像。

红场有不少著名的建筑物，南面是瓦西里升天大教堂，又名波克罗夫大教堂。它是伊凡雷帝为了纪念1552年战胜喀山鞑靼军队而下令建筑的。当你看到这座教堂，将会被它独特的建筑风格所吸引。这座教堂中间是一个带有大尖顶的教堂冠，8个带有不同色彩和花纹的小圆顶错落有致的分布在它的周围，再配上九个金色洋葱头状的教堂顶，绝妙无比。伊凡雷帝为了别处不再出现这样美丽的教堂，下令弄瞎了建筑师的双眼。

参观完红场之后，我就和amei汇合，连夜前往圣彼得堡了。

克里姆林宫

从圣彼得堡回来，克里姆林宫是我的最后一站了。

“克里姆林”在俄语中意为”内城”。在蒙古语中，是”堡垒”之意。建于1256年，位于俄罗斯首都的最中心的博罗维茨基山岗上，南临莫斯科河，西北接亚历山大罗夫斯基花园，东南与红场相连，呈三角形。保持至今的围墙长2235米，厚6米，高14米，围墙上有塔楼18座，参差错落地分布在三角形宫墙上，其中最壮观、最著名的要属带有鸣钟的救世主塔楼。5座最大的城门塔楼和箭楼装上了红宝石五角星，这就是人们所说的克里姆林宫红星。克里姆林宫享有“世界第八奇景”的美誉。

它那高大坚固的围墙和钟楼、金顶的教堂、古老的楼阁和宫殿，耸立在莫斯科河畔的博罗维茨基山岗上，构成了一组无比美丽而雄伟的艺术建筑群。莫斯科克里姆林宫是俄罗斯国家的象征，是世界上最大的建筑群之一，是历史瑰宝、文化和艺术古迹的宝库。曾为莫斯科公国和18世纪以前的沙皇皇宫。
那天下着小雨，踏进克林姆林宫的那一刻，我想，这可能是我和世界最强男人最接近的一次了，我主要参观了武器馆，珍宝馆。由于克林姆林宫不让拍照，所以这里图片少了很多 ： ）

武器馆原有一个大武器库，1720年，彼得大帝将其改建成博物馆。馆内收藏着许多珍贵文物，有历代沙皇用过的物品、美术工艺品，以及掠夺而来的战利品。这里的皇冠、神像、十字架、盔甲、礼服和餐具无不镶满宝石，仅福音书封面就嵌有26千克黄金，以及无以数计的宝石；哥登诺大帝的金御座上则镶有两千颗宝石。信步宫中宛如目睹沙皇奢侈的生活。

走在钻石馆里面，尤其是欣赏那些钻石珠宝，闪瞎了我的狗眼。

莫斯科地铁

除了这些景点，不能不提的是莫斯科地铁，全称为列宁莫斯科市地铁系统（Московский метрополитен имени В. И. Ленина），被公认为世界上最漂亮的地铁，按运营路线长度为全球第7大地铁系统。地铁站的建筑造型各异、华丽典雅。每个车站都由国内著名建筑师设计，各有其独特风格，建筑格局也各不相同，多用五颜六色的大理石，花岗岩，陶瓷和五彩玻璃镶嵌除各种浮雕，雕刻和壁画装饰，照明灯具十分别致，好像富丽堂皇的宫殿，享有“地下的艺术殿堂”之美称。

考虑到它1935年就已经开通，这里不得不惊讶于我国的地铁开发速度，我国的发展还是非常迅速的，用短短三十年赶上了走过了从前的世界强国五十年以上的发展之路，这些高速发展的背后，人民的幸福感普遍被剥夺了，在历史的进程之前，一个普通人是否快乐是否有高潮，这些都是不值得一提的。中国人过得真苦，不过欧洲人这么懒散，这么浪，迟早药丸，不是被自己玩死了，就是被绿绿玩死了。

回国

快速参观完克里姆林宫之后，我找到梁博归还他的羽绒服，由于此前不慎丢失了钱包和移民卡，我还借了梁博5000卢保命钱，最后一餐是他请我吃的日本料理，想起一天之前我还和他吹比说请他吃牛排，这多少有点讽刺。过关时候我担心移民卡丢失导致我会被滞留，直到海关姐姐说you can go now的，我放佛看到万丈佛光在她背后展开，只想跪下叫她一声爸爸。在返程起飞后不久，我就睡着了，这一次旅途所有的开心，兴奋，惊吓，沮丧等情绪全部抛在脑后，精疲力尽的我只想早点回到家乡，然后去参加总裁的婚礼。
